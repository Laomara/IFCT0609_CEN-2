## Ejemplo de app con Express

Seguimos los pasos de [este tutorial](https://medium.com/@onejohi/building-a-simple-rest-api-with-nodejs-and-express-da6273ed7ca9)

Para crear el `package.json`:
```
npm init -y 
```
Para instalar express:
```
npm install express
```
